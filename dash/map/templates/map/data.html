Data Downloads

All of the scores computed in this dashboard are open source, and you are free to download them and use them as you would like. Note that the data provided here is done so as is.

There are three types of data sets you can download for each region: Scores, summaries, and supplemental data. Scores data contains the specific score values for each block group in the entire urban area. Summary data contains population-weighted averages for both the urban area and the city core definitions themselves. Supplemental data consists of geospatial data, transit lines, and other data that might be useful when examining the results.

To retrieve the appropriate data, please select both a region and a date.
