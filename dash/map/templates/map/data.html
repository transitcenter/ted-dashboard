Data Downloads

All of the scores computed in this dashboard are open source, and you are free to download them and use them as you would like. If you use this dashboard's data in a publication please don't forget to cite us. Note that the data provided here is done so as is and with no additional warranty.

There are three types of data sets you can download for each region: Scores, summaries, and supplemental data. Scores data contains the specific score values for each block group in the entire urban area. Summary data contains population-weighted averages for both the entire urban area and the core cities themselves. Supplemental data consists of geospatial data, demographic data, transit lines, and other data that was used in the analysis.

Scores and summaries data are formated as CSV files. Supplemental data are formatted as CSV or GeoJSON files, depending on the dataset.

To retrieve the appropriate data, please select both a region and a date.
