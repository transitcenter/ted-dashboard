June 2024

Version 2.0

The 2.0 version of TED includes a number of major updates to the methdology, data analysis and storage, and website design. Here are some highlights:

New Travel Times
All travel times across all dates were recomputed using the powerful r5py library built on the R5 engine.
New Fares
Fares were recomputed for all regions and updated using 2023 fares for select regions.
Regional Definition Updates
Regional definitions were updated based on urbanized area extents, instead of entire Metropolitan Statistical Area definitons.
Tiled Maps
Data maps now use vector tilesets, hosted on Mapbox. This speeds up both data retrieval and data updates.
New Website Look
The dashboard has a new look and feel, and includes some key backend structural to make updates easier.
Additional and Refined Measures
90-minute cumulative travel time measures are added for access to jobs. Travel time access to early voting locations added. Time-period-specific transit service intensity measures added.
